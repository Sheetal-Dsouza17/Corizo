## Stock Price Prediction
**Project Overview**

This project is part of the Corizo program and focuses on predicting stock prices using machine learning techniques. The goal is to build a model that can accurately predict future stock prices based on historical data. The project involves data preprocessing, model building, training, and evaluation.

**##Wine Classification**
**Project Overview**

This project focuses on classifying wines based on their chemical properties using machine learning techniques. The goal is to build a model that can accurately classify the quality of wine. The project involves data preprocessing, model building, training, evaluation, and handling class imbalance.
